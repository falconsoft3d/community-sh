Ayuda

Guías e instrucciones para configurar la plataforma

Cómo obtener un GitHub Personal Access Token

  1. Ve a GitHub Settings → Developer settings → Personal access tokens
  2. Haz clic en "Generate new token" (classic)
  3. Dale un nombre descriptivo (ej: "Community SH")
  4. Selecciona los siguientes permisos:
    • repo - Acceso completo a repositorios privados
    • read:org - Leer información de organizaciones (opcional)
  5. Haz clic en "Generate token"
  6. Copia el token y pégalo en la pestaña GitHub (guárdalo en un lugar seguro, no podrás verlo de nuevo)

Configurar SSL con Let's Encrypt

  1. Asegúrate de que tu dominio esté apuntando a la IP de este servidor
  2. Verifica que el puerto 80 esté disponible y accesible desde internet
  3. Ve a la pestaña "Domain & SSL"
  4. Ingresa tu dominio principal
  5. Haz clic en "Generar Certificado SSL"
  6. Completa el formulario con tu dominio y email
  7. El sistema generará automáticamente el certificado y lo configurará
  8. Los certificados se renovarán automáticamente antes de expirar

Consejos generales

  • Guarda siempre los cambios después de modificar cualquier configuración
  • Mantén tu GitHub Personal Access Token seguro y no lo compartas
  • Si cambias el dominio principal, actualiza también tus registros DNS
  • El registro de usuarios puede deshabilitarse temporalmente durante mantenimiento
  • Revisa regularmente la pestaña "Acerca de" para conocer nuevas características
