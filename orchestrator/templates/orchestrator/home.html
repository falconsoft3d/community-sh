Community SH

A powerful open-source Odoo orchestration platform. Deploy, manage, and scale multiple Odoo instances with ease.

Quick Install (Linux/macOS):

curl -sSL https://raw.githubusercontent.com/falconsoft3d/community-sh/main/install.sh | bash
Get Started Sign In Star on GitHub

Multi-Instance Management

Deploy and manage multiple Odoo instances from a single dashboard. Each instance is isolated with its own database and configuration.

GitHub Integration

Connect your GitHub repositories and deploy custom Odoo modules automatically. Support for branches and webhooks.

Docker Powered

Built on Docker for maximum portability and scalability. Each instance runs in isolated containers with Traefik routing.

Backup & Restore

Create full backups of your instances including database and filestore. Restore with a single click.

Instance Duplication

Clone existing instances with all data and configurations. Perfect for testing and staging environments.

System Metrics

Monitor your infrastructure with real-time metrics for CPU, RAM, disk usage, and Docker container status.
